# Task App (Flutter)

Este repositório contém um **Task App** desenvolvido com **Flutter** e **Dart**.

O projeto foi criado como parte de uma disciplina de Desenvolvimento Mobile da faculdade.

## Status do Projeto

> Projeto educacional — sem manutenção ativa.

A disciplina terminou antes do app ser concluído, então este repositório foi mantido principalmente como referência de portfólio/arquivo.

## Stack

- Flutter
- Dart

## Observações

- Este projeto **não** representa a stack na qual pretendo focar profissionalmente hoje.
- Mesmo assim, ele está público para quem quiser explorar o código e a estrutura.
